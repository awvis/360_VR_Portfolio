# 360° Capture, render and view
## 1. 360° photo viewer

Rendering 360° photo panorama based on JS.

[Demo:](http://360.partikle.xyz/360_viewer/)

#### Features
* Pan, Zoom and Rotate

* Autorotate

* Add custom markers
#### Compatibility
* Web Browser
* Android /iOS


#### Dependencies
1. three.js
2. doT.js
3. uEvent.js
4. D.js
